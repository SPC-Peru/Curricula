\section{Objetivos de aprendizaje}

En esta malla curricular Utilizamos tres niveles de dominio, los mismos que se definen como: 
\begin{description}
 \item [Familiaridad:] El estudiante entiende lo que es un concepto o lo que significa. Este nivel de dominio se refiere a un conocimiento básico de un concepto en lugar de esperar instalación real con su aplicación. Proporciona una respuesta a la pregunta "¿Qué sabe usted acerca de esto?" 
 \item [Uso:] El alumno es capaz de utilizar o aplicar un concepto de una manera concreta. El uso de un concepto puede incluir, por ejemplo, apropiadamente usando un concepto específico en un programa, utilizando una técnica de prueba en particular, o la realización de un análisis particular. Proporciona una respuesta a la pregunta "¿Qué sabes cómo hacerlo?" 
 \item [Evaluación]: El alumno es capaz de considerar un concepto de múltiples puntos de vista y / o justificar la selección de un determinado enfoque para resolver un problema. Este nivel de dominio implica más que el uso de un concepto; se trata de la posibilidad de seleccionar un enfoque adecuado de las alternativas conocidas. Proporciona una respuesta a la pregunta "¿Por qué hiciste eso?"
\end{description}
